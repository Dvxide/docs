# DVX1DE — Portfolio Website

Personal portfolio website for a System Administrator, built with **Vite + React + TypeScript + Tailwind CSS**.

## Features

- **Boot Sequence** — Terminal-style splash screen with CRT overlay before the main site loads
- **Dark / Light Mode** — Toggle between Cyber-Noir dark and light theme, persisted in localStorage
- **Smooth Scroll Navigation** — All nav links and buttons scroll to the target section with a red highlight animation
- **Hero** — Typewriter terminal prompt, asymmetric headline, CTA buttons and stats row
- **Who I Am** — Logo with pulsing "Available for Work" badge, vertical section label, full-width statement text, glassmorphism highlight cards
- **Skill Matrix** — Filter chips, bento-grid layout, animated progress bars, automation tools overview
- **GitHub Activity** — Contribution graph (364 cells, red color scale) linking to GitHub profile
- **Projects** — Asymmetric 12-column bento grid with image-overlay and icon cards
- **Contact** — Glass-card contact methods (Email, GitHub, X/Twitter)
- **404 Page** — Terminal-aesthetic error page

## Tech Stack

| Layer | Technology |
|---|---|
| Framework | React 18 + Vite 5 |
| Language | TypeScript (strict mode) |
| Styling | Tailwind CSS v3 with CSS variable tokens |
| Fonts | Space Grotesk · Inter · JetBrains Mono |
| Icons | Material Symbols Outlined |

## Design System

- **Palette:** surface `#131317` · primary `#ffb4a8` · accent `#ff5540`
- **Aesthetic:** Terminal Prestige / Cyber-Noir — glassmorphism cards, CRT scanlines, red glow effects
- **No-Line Rule:** no hard 1px dividers — tonal background transitions only
